USS properties reference

The following table lists the USS properties in UI Toolkit.

The table indicates which properties are inherited and their animatability. For an inherited property, the element gets the value from its parent element if you don’t specify a value for it. The following example uses an inherited property to set the font for all elements.

:root {
    -unity-font: resource("Font/consola.ttf");
}

Click the property name to see detailed information about that property.

Note: The usage of a USS property that’s the same as the CSS property links to the Mozilla developer documentation.

align-content 否 Discrete Alignment of the whole area of children on the cross axis if they span over multiple lines in this container.
align-items 否 Discrete Alignment of children on the cross axis of this container.
align-self 否 Discrete Similar to align-items, but only for this specific element.
all 否 Fully animatable Allows resetting all properties with the initial keyword. Does not apply to custom USS properties.
background-color 否 Fully animatable Background color to paint in the element’s box.
background-image 否 Discrete Background image to paint in the element’s box.

border-bottom-right-radius 否 Fully animatable The radius of the bottom-right corner when a rounded rectangle is drawn in the element’s box.
border-bottom-width 否 Fully animatable Space reserved for the bottom edge of the border during the layout phase.

border-top-width 否 Fully animatable Space reserved for the top edge of the border during the layout phase.

bottom 否 Fully animatable Bottom distance from the element’s box during layout.

cursor 否 Non-animatable Mouse cursor to display when the mouse pointer is over an element.
display 否 Discrete Defines how an element is displayed in the layout.
flex 否 Fully animatable Shorthand for flex-grow, flex-shrink, flex-basis
flex-basis 否 Fully animatable Initial main size of a flex item, on the main flex axis. The final layout might be smaller or larger, according to the flex shrinking and growing determined by the other flex properties.
flex-direction 否 Discrete Direction of the main axis to layout children in a container.
flex-grow 否 Fully animatable Specifies how the item will shrink relative to the rest of the flexible items inside the same container.
flex-shrink 否 Fully animatable Specifies how the item will shrink relative to the rest of the flexible items inside the same container.
flex-wrap 否 Discrete Placement of children over multiple lines if not enough space is available in this container.
font-size 是 Fully animatable Font size to draw the element’s text.
height 否 Fully animatable Fixed height of an element for the layout.
justify-content 否 Discrete Justification of children on the main axis of this container.
left 否 Fully animatable Left distance from the element’s box during layout.
letter-spacing 是 Fully animatable Increases or decreases the space between characters.

min-width 否 Fully animatable Minimum width for an element, when it is flexible or measures its own size.

transition-duration 否 Non-animatable Time a transition animation should take to complete.
transition-property 否 Non-animatable Properties to which a transition effect should be applied.
transition-timing-function 否 Non-animatable Determines how intermediate values are calculated for properties modified by a transition effect.
translate 否 Fully animatable A translate transformation.

-unity-background-scale-mode 否 Discrete Background image scaling in the element’s box.

-unity-paragraph-spacing 是 Fully animatable Increases or decreases the space between paragraphs.

-unity-slice-scale 否 Fully animatable Scale applied to an element’s slices.
-unity-slice-top 否 Fully animatable Size of the 9-slice’s top edge when painting an element’s background image.
-unity-text-align 是 Discrete Horizontal and vertical text alignment in the element’s box.
-unity-text-outline 否 Fully animatable Outline width and color of the text.
-unity-text-outline-color 是 Fully animatable Outline color of the text.
-unity-text-outline-width 是 Fully animatable Outline width of the text.
-unity-text-overflow-position 否 Discrete The element’s text overflow position.
visibility 是 Discrete Specifies whether or not an element is visible.
white-space 是 Discrete Word wrapping over multiple lines if not enough space is available to draw the text of an element.
width 否 Fully animatable Fixed width of an element for the layout.
word-spacing 是 Fully animatable Increases or decreases the space between words.
